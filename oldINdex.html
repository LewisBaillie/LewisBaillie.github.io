About me!

My name is Lewis Baillie and I'm a student at Staffordshire University. I'm studying BSc (Hons) Computer Games Development, graduating in 2022. I'm an enthusiastic programmer with an interest in audio programming. I have experience with both FMOD and Wwise, and implementing not only audio engines but specific audio features which directly affect gameplay. Below are some of my projects, with the link to the github they are stored on.

Final Year Project, Algorithmic Beat Mapping

Github link!

For my final year research project, I decided to create an algorithmic beat mapping system. This system will be packaged for use in commercial engines so that other developers can implement a system in which they input an audio file (can also allow the end-user to do this) and the system will return a beatmap for use by the game. This system will be created using the Wwise engine and C++

Group Project, WIP-Title

In the final year of university, we are given the task of creating a game engine and making a game within that engine as a group. For the group I am part of, we will be making a survival game! I am primarily in charge of the audio implementation, sound design and sound spotting for the game. The audio implementation for this system will be utilizing the FMOD API. On top of this, I will be directly contributing to the gameplay aspects of the game.

Path Traced Audio in Unreal Engine 4

Github link!

During my final year of univeristy, there was a module in which we had to create a complicated mechanic in Unreal Engine 4. For this project, I decided to create the foundation for a path traced audio system, in the hopes of achieving a basic dynamic sound environment.
