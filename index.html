Pacman Clone

Github link!

For my semester 1 assignment for the fundamentals of games and graphical systems module, I had to make a pacman clone. This is built as a C++ native app, using a library called S2D that the university gives to new students as it allows for easy interface between OpenGL and OpenAL without prior knowledge. Other than the programming for the S2D library, all programming and asset creation was done by me.
